2. 농자제구매영수증 : 영수증에 어떤 내용이 있어야하는지? 
   - 본인이 산 영수증으로 농협에서 모종 구매한 영수증이 확실함.
     - 거래명세서 품목 나와야함 + 카드영수증 또는 현금영수증
     - 금액 기준은 없고, 품목이 중요한데 모종이면 OK
     - 실경작여부를 첨부 서류가 미비하면 등록 번거로움
       확실한 방법 >> 모종 구매하고, 거래명세서 품목 나오게.
3. 증여의경우 > 자경 농지 :> 영농사실확인서 ( 등록할 농지 소재지 이장님한테 확인 )

   - 농지대장에 경작자를 임차인으로 등록

   - 96년 이후 취득 농지는 농어촌공사를 통해서 확인 필요 << 소유자와 경작자가 따로 서류를 준비필요함
